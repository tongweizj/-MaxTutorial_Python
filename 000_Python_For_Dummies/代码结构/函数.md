# 函数

## 基本
```python
def greet_user():
    """ show simple 问候语"""    # 这叫 ‘文档字符串’，注释
    print("Hello!")

greet_user()
```

## 传递参数
```python
def greet_user(username):
    """ show simple 问候语"""  
    print(f"Hello! {username}")

greet_user('Max')
```

参数不用定义类型

### 实参和形参

```python
def greet_user(username):  #username 是形参
    """ show simple 问候语"""  
    print(f"Hello! {username}")

greet_user('Max') # ‘Max’ 是实参
```

#### 传递：位置实参

基于形参的定义顺序来传递实参


#### 传递： 关键词实参
```python
greet_user(username='Max') 
```
在dart里面，在形参位置要加上{} ,才能写成关键词实参。
但python不用，更加灵活

#### 默认值

```python
def greet_user(username, sex='man'): 
     """show simple 问候语"""  
     print(f"Hello! {username}，you are {sex}")

 greet_user('max')

def greet_user(sex='man'): 
    """show simple 问候语"""  
    print(f"Hello! you are {sex}")

greet_user()
```

#### 可选实参

在形参处提供默认值，可实现

#### 任意数量多实参

```python
def make_pizza(*toppings):
    """print customer's toppings"""
    print(toppings)

make_pizza('pepperoni')
make_pizza('mushrooms', 'green peppers')
```

*toppings 是创建一个空元组，用来接收实参
*toppings 接收的应该是相同类型的实参

**user_info  是创建一个空字典，功能和上类似
