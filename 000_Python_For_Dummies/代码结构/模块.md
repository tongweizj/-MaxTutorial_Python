# 模块

模块就是单独的文件
处理相同问题的函数放在一个模块里

## 导入

*导入方法-1*
导入整个模块

```python
import pizza #将pizza模块中的函数全部倒入
module_name.function_name() # 使用

```

*导入方法-2*
导入函数
```python
from pizza import hawaii # 只从模块中导入特定的函数
from module_name import function_0, function_1
function_0() #使用

```

*导入方法-3*

导入全部函数

```python
from module_name import *
function_0() #直接使用
```
这样的方法比方法1简单，但是如果存在相同函数名的情况，就会有问题。
并且也容易造成记忆混乱

## as 别名

```python
import pizza as pz #将pizza模块设置别名
pz.function_name() # 使用


from pizza import hawaii as hw # 给函数设置别名
hw() #使用

```
